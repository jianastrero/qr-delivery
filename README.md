## Description
This is a template Android app that uses Jetpack Compose and Kotlin. Basic Android app Template using Jetpack Compose and follows SOLID and DDD principles.

## Technologies
* Android
* Kotlin
* Jetpack Compose

## Usage
1. Click the green "Use this template" button on the top right of the repository.
2. Enter the repository name and description.
3. Click "Create repository from template".
4. Clone the repository.
5. Open the project in Android Studio.
6. Customize the app to your liking.
    - Change the package name.
    - Change the app name.
    - Change the app icon.
    - Change the app colors.
    - Change the app theme.
    - Change the app fonts.
    - Change the app splash screen.
